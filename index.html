FromThePage.org

Open-source transcription, indexing, OCR correction and translation.

FromThePage.org is the home for community development of the open-source FromThePage software. If you want to run a project on FromThePage without setting up your own server, check out FromThePage.com.

Project News

  • Release 19.6 announced June 11, 2019
  • FromThePage 19.3 announced March 15, 2019
  • Release 18.11 containing ReCAPTCHA integration is announced November7, 2019.
  • Release 18.10 containing improved lexicographical support and linked-data endpoints for subjects is announced October 25, 2019.
  • Brumfield Labs adopts calendar versioning and sets an (optional) monthly release cycle.
  • FromThePage 2.2 announced April 5, 2017
  • Documentary Editing for IIIF allows projects to be launched from works hosted at any repository supporting the IIIF standard, as well as as supporting publication in IIIF clients like the Universal Viewer or Mirador.
  • FromThePage 2.1 announced October 12, 2016
  • FromThePage now supports LaTeX for encoding mathematical and chemical formulae. Enclosing mark-up in {tex: ... :tex} allows LaTeX snippets to be embedded within any transcript.

Contributing

FromThePage source code is hosted on GitHub at https://github.com/benwbrum/fromthepage. If there's a feature you'd like to request, open up an issue in the issue tracker.

We track project development priorities in the FromThePage Waffle board

To follow your own development priorities, fork the FromThePage repository, commit your changes, and create a pull request so we can look at the code and merge it into the main project (if appropriate).

Frustrated with the documentation? We need help there too! You can start editing the wiki, or just emailing document suggestions to benwbrum@gmail.com. We're especially interested in case studies explaining the approach you took in your own project when grappling with transcription conventions, subject categories, or editorial processes. (Some material will be incorporated into online help at fromthepage.com, while other material will live on the wiki.)

Support

For questions and support, please post to the FromThePage Google Group or email benwbrum@gmail.com directly.

Customization

Need custom installation or development? Email benwbrum@gmail.com or post to the FromThePage Google Group and we'll put you in contact with a freelance software developer who can help you out.
